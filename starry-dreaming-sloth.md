# 桌面番茄钟软件 - 实施计划

## Context
用户需要一个桌面番茄钟软件。采用纯前端方案（单个 HTML 文件），双击即可在浏览器中运行，无需安装任何依赖。

## 技术方案
单个 HTML 文件，包含内嵌 CSS 和 JS，存放在 `/Users/linxiahong/pomodoro.html`。

## 功能设计

### 核心功能
- 三种模式：专注 (25分钟)、短休息 (5分钟)、长休息 (15分钟)
- 圆形进度条倒计时显示
- 开始/暂停/重置 按钮
- 模式切换
- 完成番茄数计数
- 计时结束时的浏览器通知 + 提示音

### UI 设计
- 居中卡片式布局，现代简约风格
- 顶部三个模式切换按钮（红/绿/蓝配色）
- 中间大圆形进度环，显示剩余时间
- 底部控制按钮和番茄计数
- 响应式设计，移动端和桌面端都可用

### 文件清单
- **新建**: `/Users/linxiahong/pomodoro.html` — 完整番茄钟应用

## 验证方式
1. 用浏览器打开 `pomodoro.html`
2. 点击"开始"启动 25 分钟倒计时
3. 切换到短休息/长休息模式验证时间切换
4. 等待计时结束（或修改 JS 临时缩短时间）验证通知和提示音
5. 验证暂停/继续/重置功能
